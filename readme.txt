== Installation ==

**From WordPress Admin:**

1. Go to Plugins > Add New
2. Click Upload Plugin
3. Choose the ZIP file
4. Click Install Now
5. Activate the plugin

**Manual Installation:**

1. Upload the `calendario-fechas-relevantes` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Calendar Dates in the admin menu to configure

**After Installation:**

1. Navigate to Calendar Dates in your WordPress admin menu
2. Set your calendar year
3. Customize header colors
4. Add custom relevant dates
5. Use the shortcode `[calendario_fechas]` on any page

== Frequently Asked Questions ==

= How do I display the calendar on my page? =

Use the shortcode `[calendario_fechas]` in any page or post editor.

= Can I customize the colors? =

Yes! You can customize:
- Month header gradient colors (2 colors)
- Individual date colors

= Does it include holidays? =

Yes, Spanish national holidays are included automatically. You can disable this feature in settings.

= Can I add my own dates? =

Absolutely! Go to Calendar Dates > Custom Relevant Dates and add as many dates as you need with custom descriptions and colors.

= Is it mobile-friendly? =

Yes, the calendar is fully responsive and works on all devices.

= What languages are supported? =

Currently English and Spanish. The plugin is translation-ready, so you can add your own translations.

= Can I use multiple calendars for different years? =

Yes, use the year parameter: `[calendario_fechas year="2024"]`

= Do weekends show in red automatically? =

Yes, all Saturdays and Sundays are automatically marked in red.

= Can custom dates override holidays or weekends? =

Yes! Custom dates take priority over holidays and weekend marking.
